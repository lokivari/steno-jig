2. 高頻特定單音詞 | Codes for Individual High Frequency Characters

超高頻特定單音詞：XZI（是）。
The specific character code for the high-frequency character 是 "to be" is XZI.
一、超高頻特定碼「是」的特定方法與中文小寫數字特定碼有所不同，是借用有音無字的音節碼來表示的，全書只此一個。
A) Unlike codes for numbers, which use W either in addition to the normal syllable code or replacing some part of the syllable code, the code for 是 XZI is the normal syllable code plus the empty vowel I. This is the only such code.
二、一級高頻特定單音詞的打法：X-單字音節碼，舉例：
B) 1st tier high frequency single characters are stroked using X-[syl.], as shown below.
的（X-D）、自（X-DZ）
特（X-BD）、使（X-XZ）
三、二級高頻特定單音詞的打法：W-單字音節碼，舉例：
C) 2nd tier high frequency single characters are stroked using W-[syl.], as shown below.
得（W-D）、字（W-DZ）
時（W-XZ）、次（W-BDZ）
四、三級高頻特定單音詞的打法：XW-單字音節碼，全書共10個，如下：
D) 3rd tier high frequency single characters are stroked using XW-[syl.], of which there are only ten, as shown below.
乘（XW-BZNE）、處（XW-BZU）
又（XW-IEO）、予（XW-IU）
即（XW-GI）、神（XW-XZN）
省（XW-XZNE）、唯（XW-UE）
縣（XW-XINA）、爭（XW-ZNE）
