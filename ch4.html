Briefs for common three-character phrases are as follows:

[首音節碼]/X-X

參 功能碼W | C. The Function Key, W

1. 刪除錯字 | Deleting or Undoing Strokes

在錄入過程中，難免出現擊錯鍵的情況。此時可以立即用雙手並擊W-W將最後一個字、詞刪除。
Stroking W-W deletes or undoes the most recent stroke, be it a single character or a phrase.


2. 聯詞消字定字 | Character Selection by Association

在專用文字處理系統中，爲了準確地速錄某個單音詞，可以採用如下技巧。 如需綠如單音詞「知」，可先擊略碼「知道」（聯詞），然後右手單擊W消去「道」字（消字），而留下「知」字（定字）。這種打法，叫「聯詞消字定字法」。
The following techinique can be used to quickly select a given single character without having to use function keys or the like. Taking the character 知 as an example, first stroke the brief for 知道 (Z-X), then type -W to delete the character 道 on the right, leaving just 知. This technique is called Character Selection by Association, as you can use a well known phrase associated with a particular character to output that specific character.

「聯詞消字定字法」是亞偉速錄機準確錄入技巧中最常用也是最重要的一種。
Character Selection by Association is a commonly used technique for accurate transcription, and it is certainly one of the most important.
