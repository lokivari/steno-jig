Yawei Chapter 2: Labial Consonants and 1-2 Briefs

Each initial consonant has a default vowel assigned to it, so that one need only stroke the consonant without the assigned default vowel. Consider the steno codes for the following initial consonants.


Yawei Code BBGXBXBU
Pinyin b(u)p(u)m(u)f(u)
Example Character 不鋪木副

As stroking U along with B 'b', XB 'm', or GB 'p' would be redundant, this opens up U to be used as part of the initial consonant XBU 'f'. Note that should another vowel be stroked, this default vowel is dropped, as in BA 'ba' not *'bua'.

Briefs in Yawei by and large follow a particular pattern. In this chapter we'll introduce the first of such patterns, what I will call "1-2 briefs". The formula for this brief is stroking a syllable on the left while pressing X or W on the right hand. This results in a two character phrase, with the first character having the pronunciation stroked on the left hand. Consider the examples below using the above steno codes:

B-X 不能 'buneng' "cannot" B-W 部分 'bufen' "a part"
BG-X 普遍 'pubian' "common" BG-W 普通 'putong' "normal"
XB-X 目的 'mudi' "goal" XB-W 目前 'muqian' "at present"
XBU-X 複雜 'fuza' "complicated" XBU-W 負責 'fuze' "take responsibility (for)"

As can be seen by the list above, the second syllable is not predictable, so the typist must memorize the output of each brief of this sort.

Another type of Yawei steno code that can be thought of like a brief is the specific character code, which refers to a code that outputs a specific character, and not just any character pronounced with that syllable. For example, BA could refer to 爸 "father", 把 "hold", 拔 "pull out", 八 "eight", etc., but BW refers only to the last, namely 八 'ba' "eight". In fact, all the number characters have such a code, but for now we will only introduce this one and a corresponding 1-2 brief:

BW 八 'ba' "eight" BW-W 八月 'bayue' "August"
Finally, we will introduce here a few punctuation marks and function codes. The full-width period (。) and comma (，) use the same code, but occur on different hands. Likewise the code XBW creates a new line, but when stroked on the left there is not indentation on the new line, while the new line is indented with two spaces if XBW is stroked on the right hand.

DGI- 。 -DGI ， XBW- (new line) -XBW (new line, indent two spaces)
